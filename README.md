# 💫 About Me:
Hi, I'm Architha Kanchanapally, an aspiring VLSI Design and Verification Engineer with a passion for RTL Design, Digital Design, and Functional Verification. I enjoy building hardware solutions using Verilog, SystemVerilog, UVM, and industry-standard EDA tools. My projects include a 32-bit MIPS RISC Processor, a Low-Power Approximate Multiplier, and an AI-based genetic disease detection system. I'm continuously learning, building, and exploring new technologies in the semiconductor industry while contributing to open-source and personal projects.
